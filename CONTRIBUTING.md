# Contributing to Claude Code for Marketers

Thank you for your interest in contributing! This project is open to improvements and we welcome contributions from the community.

## Ways to Contribute

### Report Issues
- Found a bug? [Open an issue](https://github.com/cc4-marketing/cc4.marketing/issues)
- Have a suggestion? We'd love to hear it

### Improve Content
- Fix typos or unclear explanations
- Add examples to lessons
- Improve the practice project scenarios

### Add Features
- New lesson content
- Additional marketing workflows
- Plugin improvements

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork locally
3. **Create a branch** for your changes
   ```bash
   git checkout -b feature/your-feature-name
   ```
4. **Make your changes**
5. **Test** by running the lessons in Claude Code
6. **Commit** with a clear message
   ```bash
   git commit -m "feat: Add new lesson on email marketing"
   ```
7. **Push** to your fork
8. **Open a Pull Request**

## Content Guidelines

### Lesson Content
- Keep explanations clear and concise
- Include practical examples
- Test all commands and code snippets
- Match the existing tone (friendly, professional)

### Code Style
- Follow existing patterns in the codebase
- Comment only when necessary
- Keep files organized

### Commit Messages
Use conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `chore:` Maintenance

## Questions?

Open an issue or reach out through the repository discussions.

Thank you for helping make this course better for marketers everywhere!
