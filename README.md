**Learn AI-powered marketing workflows with interactive lessons taught inside Claude Code itself.**

![Claude Code for Marketers](public/og-image.png)

## What is this?

A free, hands-on course where you learn Claude Code by actually using it. No videos. No PDFs. Just interactive lessons.

## How it works

1. **Download** the course materials
2. **Open** the folder in Claude Code
3. **Type** `/start-0-0` to begin
4. **Learn** as Claude guides you through each lesson

## Course Content

### Module 0: Getting Started (3 lessons)
- Introduction to Claude Code
- Installation & Setup
- Starting your first project

### Module 1: Claude Code Fundamentals (7 lessons)
- Working with files
- Your first tasks
- Understanding agents
- Custom agents
- Project memory (CLAUDE.md)
- Navigation & commands

### Module 2: Marketing Workflows (6 lessons)
- Campaign briefs
- Content strategy
- Marketing copy
- Analytics & reporting
- Competitive analysis
- SEO optimization

## What's Included

- **16 interactive lessons** via slash commands
- **Practice project** - Work as Markit agency for Planerio (B2B SaaS client)
- **3 marketing personas** - Manager Mark, Founder Fiona, Director Dana
- **Bundled plugins** - CM (Compounding Marketing) & CF (Content Factory)
- **Secret bonus lessons** - Hidden content for course completers

## Quick Start

```bash
# Clone or download
git clone https://github.com/cc4-marketing/cc4.marketing.git

# Open in Claude Code
cd cc4.marketing
claude

# Start learning
/start-0-0
```

Or download directly: **[go.mtri.me/cm](https://go.mtri.me/cm)**
